# Insurance Cost Predictor

Ứng dụng dự đoán chi phí bảo hiểm y tế bằng:

- **Lasso Regression thủ công**
- **Lasso Regression sử dụng sklearn**

---

## 1. Yêu cầu môi trường

Máy tính cần cài sẵn:

- **Anaconda**
- **Python 3.13.5** hoặc phiên bản Python phù hợp trong môi trường Conda
- **VS Code** (khuyến nghị)

---

## 2. Tạo môi trường bằng Anaconda

Mở **Anaconda Prompt** hoặc terminal trong **VS Code**, sau đó chạy:

conda create -n lasso_regression_env python=3.13.5 -y

---

## 3. Kích hoạt môi trường

conda activate lasso_regression_env

---

## 4. Cài đặt các thư viện trong requirements.txt

pip install -r requirements.txt

truy cập http://localhost:7888/ để dùng web
